Importing Files

Logos InkPen can import various vector and raster formats, preserving editability whenever possible.

Import Methods

File Menu Import

  1. Choose File > Import (⌘I)
  2. Select file(s) to import
  3. Click Open to import into current document
  4. Imported content appears on active layer

Drag and Drop

  • Drag files from Finder onto canvas
  • Drop position determines placement
  • Multiple files import as separate objects
  • Hold Option while dropping to link (not embed)

Paste from Clipboard

  • Copy vector art from other applications
  • Paste with ⌘V into InkPen
  • Maintains vector properties when possible
  • Images paste as embedded bitmaps

Vector Formats

SVG Import

Scalable Vector Graphics - Best compatibility

Supported SVG Features

  • Basic shapes (rect, circle, ellipse, polygon)
  • Paths with bezier curves
  • Groups and nested structure
  • Solid fills and strokes
  • Linear and radial gradients
  • Opacity and blend modes
  • Text (converted to outlines)
  • Clipping paths
  • Transforms (translate, rotate, scale)

SVG Limitations

  • Filters may be rasterized
  • Animations are ignored
  • JavaScript is not executed
  • External references must be embedded
  • Pattern fills converted to solid

PDF Import

Portable Document Format - Print-ready artwork

PDF Import Options

  • Page Selection: Choose which page to import
  • Crop to: Media box, crop box, or art box
  • Scale: Import at 100% or fit to page
  • Flatten Transparency: For complex effects

PDF Conversion

  • Vector paths remain editable
  • Text converted to outlines
  • Gradients preserved when possible
  • Spot colors converted to CMYK
  • Images embedded as bitmaps
  • Complex effects may be rasterized

Adobe Illustrator (.ai)

Native Illustrator format (PDF-compatible)

  • Imports as PDF if saved with compatibility
  • Preserves basic vector properties
  • Layers may be flattened
  • Effects and filters not supported
  • Best to export as SVG from Illustrator first

EPS Import

Encapsulated PostScript - Legacy format

  • Basic path support only
  • May require conversion
  • Limited color space support
  • Text converted to paths

Raster Image Formats

PNG Import

Best for images with transparency

  • Full alpha channel support
  • Lossless compression
  • Good for logos and icons
  • Larger file sizes than JPEG
  • Can be traced to vectors

JPEG Import

Best for photographs

  • Compressed file size
  • No transparency support
  • Good for backgrounds
  • Quality loss with compression
  • RGB color only

TIFF Import

Professional print images

  • Lossless compression options
  • CMYK color support
  • Alpha channel support
  • Large file sizes
  • Best for print workflows

Other Image Formats

  • GIF: Limited colors, simple transparency
  • BMP: Uncompressed, Windows bitmap
  • WebP: Modern web format
  • HEIC: Apple photo format

Import Settings

Image Handling

  • Embed: Include image data in document
  • Link: Reference external file
  • Downsample: Reduce resolution for smaller files
  • Color Profile: Convert to document profile

Scaling Options

  • Original Size: Import at 100%
  • Fit to Page: Scale to page bounds
  • Custom Scale: Enter percentage
  • Resolution: Set PPI for images

Placement

  • Center: Place at page center
  • Original Position: Use file's coordinates
  • Cursor: Click to place
  • Active Layer: Add to current layer

Import Troubleshooting

Common Issues

Missing Fonts

  • Text automatically converted to outlines
  • Install missing fonts before import
  • Or accept outline conversion

Color Shifts

  • Check color profile settings
  • Convert CMYK to RGB if needed
  • Calibrate monitor for accuracy

Complex Effects Lost

  • 3D effects not supported
  • Mesh gradients simplified
  • Filters may be rasterized
  • Export from source as SVG

Large File Sizes

  • Downsample large images
  • Link instead of embed
  • Optimize before import
  • Use appropriate format

Best Practices

Preparing Files for Import

  1. Convert text to outlines in source
  2. Expand complex effects
  3. Embed linked images
  4. Use RGB color space
  5. Save as SVG when possible

Organizing Imports

  • Create dedicated layer for imports
  • Group related imported objects
  • Name layers descriptively
  • Lock reference layers
  • Delete unused imports

Image Optimization

  • Resolution: 72 PPI for web, 300 PPI for print
  • Size: No larger than needed
  • Format: PNG for graphics, JPEG for photos
  • Compression: Balance quality vs file size

Advanced Import Features

Batch Import

  • Select multiple files in import dialog
  • Each imports as separate object
  • Automatically distributed on canvas
  • Can import entire folders

Import as Template

  • Places on locked template layer
  • Dimmed for tracing reference
  • Won't export with artwork
  • Perfect for sketches

Smart Objects

  • Link to external file
  • Updates when source changes
  • Reduces document size
  • Edit in original application
