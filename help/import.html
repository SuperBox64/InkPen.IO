Importing Files

Logos InkPen can import various vector and raster formats.

Import Methods

  • File > Import (⌘I): Select files to import
  • Drag and Drop: Drag from Finder onto canvas
  • Paste (⌘V): From clipboard

Vector Formats

SVG Import

Best compatibility for vector graphics.

  • Paths, shapes, and groups
  • Gradients and opacity
  • Text (converted to outlines)
  • Transforms preserved

PDF Import

Print-ready artwork.

  • Vector paths remain editable
  • Choose page to import
  • Crop box options

AI/EPS Import

Adobe formats (limited support).

  • Basic path support
  • Best to export as SVG first

Image Formats

  • PNG: Transparency support
  • JPEG: Photographs
  • TIFF: Print quality
  • GIF/WebP/HEIC: Also supported

Import Settings

  • Embed vs Link: Include data or reference file
  • Scale: Original, fit to page, or custom
  • Placement: Center, original position, or cursor

Troubleshooting

  • Missing fonts: Text converted to outlines
  • Color shifts: Check color profiles
  • Complex effects: May be rasterized
