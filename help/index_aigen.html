Help & Documentation

Getting Started

Welcome to Logos InkPen, a professional vector art and design application for macOS. This guide will help you get started with the application.

System Requirements

  • macOS 14, 15, or 16
  • Apple Silicon or Intel processor
  • 4GB RAM minimum (8GB recommended)
  • 500MB available disk space
  • Optional: Wacom tablet for enhanced drawing experience
  • Optional: Apple Pencil support via Luna Display or Apple Sidecar

Installation

1. Download the latest version from our website

2. Open the downloaded logos.inkpen.io-1.0b9.zip file

3. Drag the InkPen app to your Applications folder

4. Launch InkPen from your Applications folder

Note: The app is signed by Apple for your security.

Basic Tools

Selection Tool

Use the selection tool to select, move, and transform objects. Click on any object to select it, or drag to create a selection rectangle.

Pen Tool

Create precise vector paths with the pen tool. Click to create corner points, or click and drag to create smooth curves.

Brush Tool

The brush tool now includes pressure sensitivity support for Apple Pencil and Wacom tablets. The new Liquid setting provides smooth, natural brush strokes.

Shape Tools

Quickly create rectangles, ellipses, polygons, and other geometric shapes. Hold Shift while dragging to constrain proportions.

Advanced Features

Layers

Organize your artwork with layers. Create, delete, reorder, and adjust layer properties from the Layers panel.

Color Management

InkPen supports RGB and CMYK color modes. Use the color picker to select colors, or create custom swatches for your projects.

Export Options

Export your work in various formats including:

  • SVG for scalable vector graphics
  • PDF for print-ready documents
  • PNG for web graphics with transparency
  • JPEG for photographs and web images

Keyboard Shortcuts

  • ⌘N - New document
  • ⌘O - Open document
  • ⌘S - Save document
  • ⌘Z - Undo
  • ⌘⇧Z - Redo
  • V - Selection tool
  • P - Pen tool
  • B - Brush tool
  • Space - Pan view (hold)
  • ⌘+ - Zoom in
  • ⌘- - Zoom out

Legacy

InkPen was inspired by Macromedia FreeHand MX and contains many of its core features. We've reimagined the classic vector design experience for modern macOS, built on SwiftUI, Metal, and CoreGraphics.
