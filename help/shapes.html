Creating Shapes

Logos InkPen provides a comprehensive set of shape tools for creating geometric objects. Each shape tool has unique properties and modifiers for precise control.

Basic Shape Creation

Drawing Shapes

  1. Select a shape tool from the vertical toolbar
  2. Click and drag on the canvas to create the shape
  3. Release the mouse to complete the shape
  4. The shape adopts current fill and stroke settings

Shape Modifiers While Drawing

  • Shift: Constrain proportions (creates squares from rectangles, circles from ellipses)
  • Option (⌥): Draw from center outward instead of corner to corner
  • Shift + Option: Constrained shape from center
  • Space: Reposition shape while drawing (hold and drag)

Rectangle Shapes

Rectangle Tool (⌥R)

Creates rectangular shapes with independent width and height.

  • Drag diagonally to define opposite corners
  • Click once to enter exact dimensions
  • Corner radius can be adjusted after creation

Square Tool (⌥S)

Creates perfect squares without needing to hold Shift.

  • Automatically constrains to equal width and height
  • Diagonal drag distance determines size
  • Ideal for icons and grid-based designs

Rounded Rectangle Tool (⇧⌥R)

Creates rectangles with pre-set rounded corners.

  • Default corner radius: 10pt
  • Adjust radius in properties panel
  • Each corner can be adjusted independently
  • Perfect for buttons and UI elements

Pill Tool (⇧⌥P)

Creates stadium/capsule shapes with fully rounded ends.

  • Corner radius automatically set to half the height
  • Cannot adjust corner radius (always maximum)
  • Great for tags, badges, and pill buttons

Circular Shapes

Circle Tool (⌥C)

Creates perfect circles without constraint keys.

  • Radius determined by drag distance
  • Hold Option to draw from center
  • Click to enter exact diameter

Ellipse Tool (E)

Creates oval shapes with variable proportions.

  • Width and height can differ
  • Hold Shift to constrain to circle
  • Useful for perspective circles and organic shapes

Oval Tool (O)

Creates elongated ovals with specific aspect ratios.

  • Optimized for common oval proportions
  • 2:1 aspect ratio by default
  • Good for pills, race tracks, and badges

Egg Tool (⇧E)

Creates egg-shaped objects with asymmetric curves.

  • Wider at bottom, narrower at top
  • Natural organic appearance
  • Perfect for character bodies and natural forms

Cone Tool (⇧⌥C)

Creates teardrop or cone shapes.

  • Rounded base with pointed top
  • Adjust roundness with handles
  • Ideal for location pins and droplets

Triangle Shapes

Equilateral Triangle (⇧T)

All three sides equal length, 60° angles.

  • Perfect geometric triangle
  • Rotates from base by default
  • Hold Shift for point-up orientation

Isosceles Triangle (I)

Two equal sides with symmetry.

  • Adjust height independently of base
  • Creates arrows and directional indicators
  • Mirror-symmetric shape

Right Triangle (⇧⌥R)

One 90-degree angle.

  • Perfect for geometric constructions
  • Corner indicators and brackets
  • Pythagorean proportions available

Acute Triangle (⇧A)

All angles less than 90 degrees.

  • Sharp, pointed appearance
  • Dynamic and directional
  • Good for arrows and pointers

Polygon Shapes

Pentagon Tool (5)

Regular 5-sided polygon.

  • Home plate shape when rotated
  • Military insignia and badges
  • 72° rotation between vertices

Hexagon Tool (6)

Regular 6-sided polygon.

  • Honeycomb and tile patterns
  • Efficient packing shape
  • 60° rotation between vertices

Heptagon Tool (7)

Regular 7-sided polygon.

  • Unique and uncommon shape
  • Badge and coin designs
  • 51.43° rotation between vertices

Octagon Tool (8)

Regular 8-sided polygon.

  • Stop sign shape
  • Classic floor tile pattern
  • 45° rotation between vertices

Nonagon Tool (9)

Regular 9-sided polygon.

  • Complex geometric shape
  • Decorative and ornamental
  • 40° rotation between vertices

Custom Polygon Tool (⌥P)

Create polygons with 3-100 sides.

  • Set number of sides before drawing
  • Automatic regular spacing
  • Can create near-circles with many sides

Star Shapes

Star Tool (⇧S)

Creates star shapes with customizable points.

Star Variants (Long-press to access)

  • 3-Point Star: Triangle-based star
  • 4-Point Star: Diamond/compass star
  • 5-Point Star: Classic star shape
  • 6-Point Star: Star of David style
  • 7-Point Star: Mystical/badge star
  • 8-Point Star: Starburst shape
  • Custom Points: 3-100 points

Star Properties

  • Outer Radius: Distance to points
  • Inner Radius: Distance to valleys
  • Ratio: Inner/outer relationship (0.3-0.9)
  • Rotation: Starting angle of first point

Shape Editing

After Creating a Shape

  • Selection Tool (A): Move and transform entire shape
  • Direct Selection (D): Edit individual points
  • Corner Radius (⌃↑): Round rectangle corners
  • Scale Tool (S): Resize proportionally or freely
  • Rotate Tool (R): Rotate around center or custom point

Converting Shapes

  • To Path: Convert shape to editable bezier path
  • Expand: Convert stroke to filled shape
  • Outline: Create path from shape outline
  • Compound Path: Combine multiple shapes with holes

Shape Properties Panel

Access Paint panel (⌘⇧P) to adjust:

  • Fill: Interior color, gradient, or pattern
  • Stroke: Outline width, color, and style
  • Opacity: Transparency of fill and stroke
  • Shadow: Drop shadow effects
  • Blend Mode: How shape interacts with objects below

Precise Shape Creation

Numeric Input

Click once with any shape tool to enter exact dimensions:

  • Width: Horizontal size in current units
  • Height: Vertical size in current units
  • Constrain: Lock aspect ratio
  • From Center: Create from click point

Smart Guides

Automatic alignment assistance while drawing:

  • Snap to other objects' edges and centers
  • Show when dimensions match existing shapes
  • Indicate equal spacing between objects
  • Display measurement labels

Grid Snapping

Enable grid (⌘') for precise placement:

  • Shapes snap to grid intersections
  • Adjust grid spacing in preferences
  • Hold ⌘ to temporarily disable snapping

Shape Tips and Tricks

Pro Tips

  • Double-click shape tools to set default properties
  • Option-drag existing shapes to duplicate
  • Use Transform Each to modify multiple shapes
  • Group similar shapes for easier management
  • Lock shapes to prevent accidental changes
  • Use layers to organize complex compositions

Common Shape Workflows

Creating Icons

  1. Start with basic geometric shapes
  2. Use boolean operations to combine
  3. Apply consistent corner radius
  4. Maintain stroke width throughout

Building Patterns

  1. Create base shape
  2. Duplicate with Option-drag
  3. Use Distribute to space evenly
  4. Group and duplicate pattern tile

Logo Construction

  1. Build with simple shapes first
  2. Refine with Direct Selection tool
  3. Unite shapes for single color areas
  4. Convert to outlines when complete
