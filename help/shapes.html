Creating Shapes

Logos InkPen provides a comprehensive set of shape tools for creating geometric objects.

Basic Shape Creation

  1. Select a shape tool from the toolbar
  2. Click and drag on the canvas
  3. Release to complete the shape

Shape Modifiers

  • Shift: Constrain proportions
  • Option (⌥): Draw from center
  • Space: Reposition while drawing

Rectangle Shapes

  • Rectangle (⌥R): Variable width and height
  • Square (⌥S): Equal sides
  • Rounded Rectangle (⇧⌥R): With corner radius
  • Pill (⇧⌥P): Fully rounded ends

Circular Shapes

  • Circle (⌥C): Perfect circles
  • Ellipse (E): Ovals
  • Egg (⇧E): Asymmetric ovals
  • Cone (⇧⌥C): Teardrop shapes

Triangle Shapes

  • Equilateral (⇧T): All sides equal
  • Isosceles (I): Two equal sides
  • Right Triangle: One 90° angle
  • Acute (⇧A): All angles under 90°

Polygons

  • Pentagon (5): 5 sides
  • Hexagon (6): 6 sides
  • Heptagon (7): 7 sides
  • Octagon (8): 8 sides
  • Custom (⌥P): 3-100 sides

Star Tool (⇧S)

Create stars with customizable points (3-100) and adjustable inner/outer radius ratio.

Shape Editing

  • Selection (A): Move and transform
  • Direct Selection (D): Edit points
  • Corner Radius (⌃↑): Round corners
  • Scale (S): Resize
  • Rotate (R): Rotate

Shape Properties

Use Paint panel (⌘⇧P) to adjust:

  • Fill color and opacity
  • Stroke width and style
  • Shadow effects
  • Blend modes
