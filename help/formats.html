File Formats

Logos InkPen supports three primary formats plus specialized export options.

InkPen Format (.inkpen)

Native format - preserves all editing capabilities.

  • Maintains all layers
  • Keeps text editable
  • Preserves gradients
  • Smallest file size
  • JSON-based, compressed

Use for: Working files, templates, archiving

PDF Format (.pdf)

Industry standard for print.

  • Vector graphics scalable
  • Text converted to outlines
  • Universal compatibility
  • Print-ready output

Use for: Print shops, presentations, sharing

SVG Format (.svg)

Scalable Vector Graphics for web.

  • XML-based vector format
  • Infinitely scalable
  • Small file sizes
  • Browser compatible
  • Editable in code

Use for: Web graphics, icons, animations

PNG Format (.png)

Raster images with transparency.

  • Full alpha channel
  • Lossless compression
  • Resolution dependent

Use for: App icons, web images, screenshots
