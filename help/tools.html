Drawing Tools

Logos InkPen provides a comprehensive set of professional drawing and editing tools. Access tools from the vertical toolbar on the left side of the window or use keyboard shortcuts.

Pro Tip: Hold Shift while drawing to constrain proportions. Hold Option to draw from center.

Selection & Navigation Tools

Selection Tool (A)

The primary tool for selecting, moving, and transforming complete objects.

  • Click on any object to select it
  • Drag to move objects
  • Click and drag from empty space for marquee selection
  • Hold Shift to add/remove from selection
  • Double-click for isolation mode

Direct Selection Tool (D)

Fine-tune shapes by manipulating individual anchor points and bezier handles.

  • Click on a path to reveal anchor points
  • Drag anchor points to reshape
  • Adjust bezier handles for curves
  • Shift-click to select multiple points

Hand Tool (H or Space)

Pan around the canvas. Hold Space for temporary hand tool while using other tools.

Zoom Tool (Z)

Click to zoom in, Option-click to zoom out. Use ⌘+ / ⌘- for quick zoom.

Path Drawing Tools

Bezier Pen Tool (P)

Create precise vector paths for logos, custom lettering, and technical illustration.

  • Click for corner points
  • Click and drag for smooth curves
  • Click first point to close path
  • Escape to end open path

Freehand Tool (F)

Quick sketching with automatic smoothing. Draw naturally with mouse or tablet.

Brush Tool (B)

Create artistic strokes with variable width. Supports pressure sensitivity.

Marker Tool (M)

Marker-style stroke simulating a chisel-tip marker.

Transform Tools

Scale Tool (S)

Resize objects. Hold Shift for proportions, Option for center scaling.

Rotate Tool (R)

Rotate objects. Hold Shift for 45° increments.

Shear Tool (X)

Skew objects for italic or perspective effects.

Warp Tool (W)

Distort objects with envelope warping.

Shape Tools

Line Tool (L)

Draw straight lines. Hold Shift for 45° angles.

Rectangle Tool (⌥R)

Create rectangles. Hold Shift for squares, Option for center drawing.

Circle Tool (⌥C)

Draw perfect circles.

Ellipse Tool (E)

Create ovals. Hold Shift for circles.

Polygon Tool (⌥P)

Regular polygons with 3-100 sides.

Star Tool (⇧S)

Stars with customizable points and inner radius.

Triangle Tools

  • Equilateral (⇧T): All sides equal
  • Isosceles (I): Two equal sides
  • Right Triangle: One 90° angle
  • Acute (⇧A): All angles under 90°

Specialty Tools

Text Tool (T)

Add and edit text. Click for point text, drag for area text.

Eyedropper Tool (I)

Sample colors from any object.

Gradient Tool (G)

Apply and adjust gradients on selected objects.

Corner Radius Tool (⌃↑)

Dynamically round corners on rectangles and shapes.

Tool Modifiers

  • Shift: Constrain proportions or angles
  • Option (⌥): Draw from center or duplicate
  • Command (⌘): Temporarily switch to Selection tool
  • Space: Temporarily switch to Hand tool
