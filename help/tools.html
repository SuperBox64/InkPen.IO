Drawing Tools

Logos InkPen provides a comprehensive set of professional drawing and editing tools. Access tools from the vertical toolbar on the left side of the window or use keyboard shortcuts for faster workflow.

Pro Tip: Hold Shift while drawing to constrain proportions. Hold Option to draw from center.

Selection & Navigation Tools

Selection Tool

ShortcutA
IconArrow cursor

Purpose: The primary tool for selecting, moving, and transforming complete objects. This is your most-used tool for general editing.

How to use:

  • Click on any object to select it - selection handles appear around the object
  • Drag an object to move it to a new position
  • Click and drag from empty space to create a marquee selection box
  • Hold Shift and click to add or remove objects from selection
  • Double-click to enter isolation mode for groups
  • Drag corner handles to resize selected objects

Best for: General object manipulation, arranging layouts, selecting multiple objects for group operations.

Direct Selection Tool

ShortcutD
IconWhite arrow

Purpose: Fine-tune the shape of paths by manipulating individual anchor points and bezier handles. Essential for precise path editing.

How to use:

  • Click on a path to reveal all anchor points
  • Click and drag an anchor point to move it
  • Click a point to select it, then drag the bezier handles to adjust curves
  • Shift-click to select multiple anchor points
  • Drag a path segment between points to reshape the curve

Best for: Refining logos, adjusting letterforms, perfecting curves, fixing path problems.

Hand Tool

IconHand

Pan around the canvas without changing the zoom level. Hold Space bar for temporary hand tool while using any other tool.

Path Drawing Tools

Bezier Pen Tool

ShortcutP
IconPen nib

Purpose: The professional's choice for creating precise vector paths. Essential for logo design, custom lettering, and technical illustration.

How to use:

  • Corner points: Click once to place a sharp corner point
  • Smooth curves: Click and drag to pull out bezier handles that define the curve
  • Continue path: Keep clicking to add more points to your path
  • Close path: Click on the first point to create a closed shape
  • End path: Press Escape or switch tools to finish an open path
  • Convert points: Hold Option and click a smooth point to convert to corner

Pro tips:

  • Use as few points as possible for smoother curves
  • Place points at the extremes of curves (top, bottom, leftmost, rightmost)
  • Pull handles in the direction of the curve for natural flow

Best for: Logo creation, custom typography, technical drawings, tracing artwork.

Freehand Tool

ShortcutF
IconScribble

Purpose: Quick sketching and organic drawing. Perfect for initial concept work and natural, flowing shapes.

How to use:

  • Click and drag to draw freeform paths
  • Release to complete the path - it automatically smooths
  • Adjust smoothing amount in properties before drawing
  • Draw slowly for more accurate paths
  • Draw quickly for more gestural, loose strokes

Best for: Quick sketches, organic shapes, handwritten signatures, concept drawings, natural elements like leaves or clouds.

Brush Tool

ShortcutB
IconVariable scribble

Purpose: Create artistic strokes with variable width. Simulates traditional brush techniques digitally, with full pressure sensitivity support.

How to use:

  • Select brush size and smoothness in the properties panel
  • Draw with mouse or stylus - pressure affects stroke width (with tablet)
  • Lighter pressure creates thin strokes, heavier creates thick strokes
  • Overlap strokes to build up artistic effects
  • Use with opacity settings for watercolor-like effects

Settings:

  • Size: Base width of the brush stroke
  • Smoothness: How much the stroke is refined after drawing
  • Pressure: Sensitivity to tablet pressure (if available)

Best for: Illustrations, calligraphy, artistic logos, hand-lettering, expressive drawings.

Marker Tool

ShortcutM
IconPencil tip

Draw with a marker-style stroke that simulates a chisel-tip marker. Great for sketching and annotations.

Convert Anchor Point Tool

ShortcutC
IconChevron

Convert between smooth and corner anchor points. Click a smooth point to convert to corner, drag a corner point to convert to smooth.

Rotate Tool

ShortcutR

ShortcutX

ShortcutW

Shape Tools

Line Tool

ShortcutL
IconDiagonal line

Draw straight lines. Hold Shift to constrain to 45° increments.

Rectangle Tool

Shortcut⌥R
IconRectangle

Purpose: Create rectangular shapes for buttons, frames, backgrounds, and structural elements in designs.

How to use:

  • Click and drag diagonally to create a rectangle
  • Hold Shift while dragging to constrain to a square
  • Hold Option to draw from the center outward
  • Hold Shift + Option for a square from center
  • Click once to enter exact dimensions numerically

After creating:

  • Use Corner Radius tool to round corners
  • Apply gradients for button effects
  • Use as clipping masks for images

Best for: UI elements, business cards, frames, backgrounds, grid layouts, app icons.

Square Tool

Shortcut⌥S
IconSquare

Purpose: Quickly create perfect squares without holding modifier keys. Ideal for icons and grid-based designs.

How to use:

  • Drag in any direction - always creates a perfect square
  • Size is determined by drag distance
  • Hold Option to draw from center
  • No need to hold Shift like with Rectangle tool

Best for: App icons, social media graphics, QR code backgrounds, checkerboard patterns, Instagram posts.

Specialty Tools

Gradient Tool

ShortcutG
IconGradient circle

Purpose: Apply and edit gradient fills on objects. Create smooth color transitions for professional effects like buttons, backgrounds, and 3D-like appearances.

How to use:

  • Select an object first
  • Click and drag across the object to define gradient direction
  • Start point = first color, end point = last color
  • Drag longer for gradual transitions, shorter for sharp transitions
  • Re-drag to adjust angle and length

Gradient types:

  • Linear: Straight color transition in one direction
  • Radial: Circular transition from center outward

Pro workflow:

  1. Apply gradient with the tool
  2. Open Gradient panel (⌘⇧G) to edit colors
  3. Add color stops for complex gradients
  4. Adjust opacity for each color stop

Best for: Buttons with depth, metallic effects, sky backgrounds, glossy surfaces, modern UI elements.

Corner Radius Tool

Shortcut⌃↑
IconRounded corner

Purpose: Dynamically adjust corner rounding on rectangles and other shapes. Create everything from slightly rounded buttons to pill shapes.

How to use:

  • Select a rectangle or shape with corners
  • Switch to Corner Radius tool
  • Drag any corner handle inward to round it
  • Drag outward to reduce rounding
  • Hold Shift to round all corners equally
  • Click individual corners to round them separately

Advanced features:

  • Asymmetric corners: Round each corner differently for unique shapes
  • Live adjustment: See changes in real-time as you drag
  • Precise values: Enter exact radius in properties panel
  • Maximum radius: Automatically limits to create pill/stadium shapes

Common uses:

  • Modern UI buttons and cards
  • App icons (different radius per platform guidelines)
  • Speech bubbles with selective corner rounding
  • Badges and tags

Best for: iOS/Android UI design, modern web elements, friendly approachable logos, notification badges.

Tool Modifiers

Universal Modifiers

  • Shift: Constrain proportions or angles
  • Option (⌥): Draw/transform from center or duplicate
  • Command (⌘): Temporarily switch to Selection tool
  • Space: Temporarily switch to Hand tool

Tool Tips

  • Double-click tool icons to access tool-specific options
  • Group frequently used tools by customizing your toolbar
  • Use number keys for quick polygon sides (3-9)
  • Press the tool shortcut again to cycle through tool variants
  • Hold down a tool to reveal related tools in a flyout menu
