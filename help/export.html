Exporting Your Work

Export your designs in various formats for different purposes - web, print, or further editing.

Export Methods

Quick Export (Toolbar)

  • Click PNG export button in toolbar
  • Uses last export settings
  • Exports to same location as document
  • Fastest method for proofs

File Menu Export

  • Export SVG: ⌘E
  • Export PDF: ⌘⇧E
  • Export PNG: ⌃⌘E
  • Export AutoDesk SVG: ⌘⌥E (96 DPI)

Save As Different Format

  1. Choose File > Save As (⌘⇧S)
  2. Select format from dropdown
  3. Choose location and name
  4. Click Save

SVG Export

SVG Options

Best for: Web graphics, scalable icons, further editing

SVG Profiles

  • SVG 1.1: Maximum compatibility
  • SVG Tiny: Mobile devices
  • SVG Basic: Reduced features
  • Compressed SVG (.svgz): Smaller file size

Text Handling

  • Convert to Outlines: Maximum compatibility, not editable
  • Embed Fonts: Preserves editability (limited support)
  • Use System Fonts: Smallest size, requires font installation

Optimization Settings

  • Decimal Places: 1-3 for web, 4+ for precision
  • Remove Hidden: Delete invisible objects
  • Merge Paths: Combine similar paths
  • Clean IDs: Remove unnecessary attributes
  • Minify: Remove whitespace for smaller files

Advanced SVG Options

  • CSS Properties: Style as attributes or CSS
  • Include XMP: Metadata for asset management
  • Responsive: Scale to container
  • Preserve Illustrator Editing: Round-trip compatibility

AutoDesk SVG (96 DPI)

Special export for AutoDesk applications:

  • Forces 96 DPI resolution
  • Ensures proper scaling in AutoCAD
  • Maintains dimensional accuracy
  • Use for technical drawings

PDF Export

PDF Settings

Best for: Print, document sharing, archival

PDF Presets

  • High Quality Print: 300 DPI, CMYK, print-ready
  • Press Quality: Maximum quality for professional printing
  • PDF/X-1a: Print industry standard
  • Smallest File Size: Web viewing, email
  • Standard: Balanced quality and size

Compression Options

  • None: No compression (largest files)
  • JPEG: Lossy compression for images
  • JPEG 2000: Better quality at same size
  • ZIP: Lossless compression
  • Automatic: Let system choose

Color Management

  • Color Space: RGB, CMYK, or Gray
  • Profile: sRGB, Adobe RGB, custom
  • Convert to: Target color space
  • Preserve Numbers: Don't convert colors

Security Settings

  • Password Protection: Require password to open
  • Permissions: Restrict printing, copying, editing
  • Encryption: 128-bit or 256-bit AES
  • Digital Signature: Certify document

PNG Export

PNG Options

Best for: Web images, screenshots, transparency

Resolution Settings

  • Screen (72 PPI): Web and screen display
  • Medium (150 PPI): Higher quality web
  • High (300 PPI): Print quality
  • Custom: Enter specific PPI

Size Options

  • Original Size: 1x scale
  • Scale: 0.5x to 4x
  • Width/Height: Specific pixel dimensions
  • Constrain Proportions: Maintain aspect ratio

Transparency

  • Transparent Background: Preserve alpha channel
  • White Background: Opaque white
  • Custom Color: Choose matte color
  • Checkerboard: Show transparency pattern

Color Options

  • 24-bit RGB: Full color
  • 8-bit Indexed: Limited palette for smaller files
  • Grayscale: Black and white only
  • Include Profile: Embed color profile

Batch Export

Export Multiple Formats

  1. Set up export presets for each format
  2. Choose File > Export All
  3. Select formats to export
  4. Choose destination folder
  5. Click Export

Asset Export

Export individual objects as separate files:

  • Select objects to export
  • Right-click > Export Selection
  • Choose format and settings
  • Each object saves as separate file
  • Names based on layer names

Artboard Export

Export multiple artboards:

  • Each artboard as separate file
  • Or all artboards in single PDF
  • Custom range selection
  • Different formats per artboard

Export Presets

Creating Presets

  1. Configure export settings
  2. Click Save Preset button
  3. Name the preset descriptively
  4. Access from preset menu

Common Presets

Web Icon Set

  • PNG format
  • Multiple sizes: 16, 32, 64, 128px
  • Transparent background
  • sRGB color space

Print Logo

  • PDF with outlines
  • CMYK color space
  • 300 DPI resolution
  • Include bleed

Social Media

  • PNG or JPEG
  • Platform-specific sizes
  • RGB color space
  • Optimized file size

Export Best Practices

File Naming

  • Use descriptive names
  • Include version numbers
  • Add size suffix (logo_128px)
  • Avoid spaces and special characters
  • Use consistent conventions

Quality vs File Size

  • Web: Balance quality with load time
  • Print: Maximum quality priority
  • Email: Keep under 10MB
  • Archive: Lossless formats

Color Consistency

  • Embed color profiles
  • Convert to sRGB for web
  • Use CMYK for print
  • Test on target devices
  • Calibrate monitors

Troubleshooting Exports

Common Issues

Blurry Exports

  • Increase resolution/PPI
  • Export at larger size
  • Use vector format (SVG/PDF)
  • Check anti-aliasing settings

Large File Sizes

  • Optimize images before export
  • Use appropriate compression
  • Remove hidden objects
  • Simplify complex paths

Color Changes

  • Check color space conversion
  • Embed color profile
  • Use absolute colorimetric
  • Preview before export

Missing Elements

  • Check layer visibility
  • Include hidden layers option
  • Verify artboard bounds
  • Check clipping masks

Platform-Specific Exports

iOS App Icons

  • Multiple sizes required
  • No transparency allowed
  • Square format (system rounds corners)
  • Include @2x and @3x versions

Android Assets

  • Multiple DPI versions
  • 9-patch for stretchable
  • Vector drawables preferred
  • Material Design guidelines

Web Favicons

  • ICO format for compatibility
  • Multiple sizes in one file
  • PNG for modern browsers
  • SVG for scalable icons
