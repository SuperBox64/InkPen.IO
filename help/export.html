Exporting Your Work

Export your designs for web, print, or further editing.

Export Methods

  • Export SVG (⌘E): Web graphics
  • Export PDF (⌘⇧E): Print and sharing
  • Export PNG (⌃⌘E): Raster images
  • AutoDesk SVG (⌘⌥E): CAD compatibility

SVG Export

Best for web graphics and scalable icons.

  • Text: Convert to outlines or embed fonts
  • Optimization: Minify, merge paths
  • Decimal places: 1-3 for web

PDF Export

Best for print and document sharing.

  • Presets: High Quality, Press, Smallest
  • Color: RGB, CMYK, or Grayscale
  • Security: Password protection

PNG Export

Best for web images with transparency.

  • Resolution: 72-300 PPI
  • Scale: 0.5x to 4x
  • Background: Transparent or solid

AutoDesk SVG (96 DPI)

For Fusion 360 and AutoCAD.

  • Forced 96 DPI resolution
  • 1.33333× scale factor
  • Ensures dimensional accuracy

Best Practices

  • Always save .inkpen first
  • Use descriptive filenames
  • Match color space to purpose
  • Test on target devices
