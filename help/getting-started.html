Getting Started with Logos InkPen

Welcome to Logos InkPen! This guide will help you create your first design.

Creating a New Document

When you launch Logos InkPen, you'll see the Document Setup window where you can configure your canvas:

  1. Choose a Canvas Size from the presets (Letter, A4, Web, Icon) or enter custom dimensions
  2. Select the Units (pixels, points, inches, or millimeters)
  3. Set the Background Color using the color picker
  4. Click Create Document to start designing
Note: You can also create a new document at any time by choosing File > New from the menu bar.

Understanding the Workspace

The Logos InkPen workspace consists of several key areas:

Tools Panel (Left)

  • Selection Tool (V): Select and move objects
  • Rectangle Tool (R): Draw rectangles and squares
  • Ellipse Tool (E): Create circles and ellipses
  • Pen Tool (P): Draw custom paths and curves
  • Text Tool (T): Add and edit text
  • Hand Tool (H): Pan around the canvas

Properties Panel (Right)

Shows properties for the selected object:

  • Position and size controls
  • Fill and stroke options
  • Opacity and blend modes
  • Shadow and effects settings

Layers Panel (Right)

Manage your design's layer structure:

  • Create new layers
  • Reorder layers by dragging
  • Show/hide layers
  • Lock layers to prevent changes

Creating Your First Shape

  1. Select the Rectangle Tool from the toolbar (or press R)
  2. Click and drag on the canvas to draw a rectangle
  3. Hold Shift while dragging to create a perfect square
  4. Use the Properties panel to adjust the fill color, stroke, and other properties

Saving Your Work

Logos InkPen supports multiple file formats:

Native Format (.inkpen)

Save your work in the native InkPen format to preserve all layers, effects, and editing capabilities:

  • Choose File > Save (⌘S)
  • Select "Ink Pen Document" from the format dropdown
  • Name your file and choose a location

Export Formats

Export your finished designs to share or use in other applications:

  • SVG: Scalable vector format for web use
  • PDF: Universal document format for print and sharing
  • PNG: Raster image with transparency support
Important: Always save your work in the native .inkpen format first to preserve all editing capabilities. Export formats are for final output only.

Keyboard Shortcuts

Speed up your workflow with these essential shortcuts:

  • ⌘N: New document
  • ⌘O: Open document
  • ⌘S: Save document
  • ⌘Z: Undo
  • ⌘⇧Z: Redo
  • ⌘C: Copy
  • ⌘V: Paste
  • ⌘D: Duplicate
  • Delete: Delete selected object
