Keyboard Shortcuts

Master these keyboard shortcuts to work faster in Logos InkPen.

File Operations

⌘NNew Document
⌘OOpen Document
⌘SSave
⌘⇧SSave As
⌘WClose Window
⌘QQuit Application

Edit Operations

⌘ZUndo
⌘⇧ZRedo
⌘XCut
⌘CCopy
⌘VPaste
⌘DDuplicate
⌘ASelect All
⌘⇧ADeselect All
DeleteDelete Selected

Tool Selection

VSelection Tool
RRectangle Tool
EEllipse Tool
LLine Tool
PPen Tool
TText Tool
HHand Tool (Pan)
ZZoom Tool

View Controls

⌘+Zoom In
⌘-Zoom Out
⌘0Fit to Window
⌘1Actual Size (100%)
⌘'Show/Hide Grid
⌘;Show/Hide Guides
⌘RShow/Hide Rulers

Object Arrangement

⌘⇧]Bring to Front
⌘]Bring Forward
⌘[Send Backward
⌘⇧[Send to Back
⌘GGroup
⌘⇧GUngroup
⌘LLock/Unlock

Alignment

⌘⌥LAlign Left
⌘⌥CAlign Center (Horizontal)
⌘⌥RAlign Right
⌘⌥TAlign Top
⌘⌥MAlign Middle (Vertical)
⌘⌥BAlign Bottom

Transform Operations

⌘TFree Transform
⌘⌥TTransform Again
⌘⇧HFlip Horizontal
⌘⇧VFlip Vertical

Path Operations

⌘JJoin Paths
⌘⌥JAverage Points
⌘⇧OCreate Outlines
⌘⌥CMake Compound Path
⌘⌥⇧CRelease Compound Path

Boolean Operations

⌘⌥UUnite (Union)
⌘⌥SSubtract (Minus Front)
⌘⌥IIntersect
⌘⌥XExclude (XOR)

Special Keys

SpaceTemporarily activate Hand Tool
ShiftConstrain proportions/angles
OptionDuplicate while dragging
⌘SpaceZoom In (temporary)
⌘⌥SpaceZoom Out (temporary)
Tip: Hold down Shift while drawing to create perfect squares, circles, and straight lines. Hold Option while dragging to duplicate objects.

Customizing Shortcuts

You can customize keyboard shortcuts in Logos InkPen > Preferences > Keyboard. Changes take effect immediately and are saved automatically.
