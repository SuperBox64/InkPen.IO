Panels Reference

Logos InkPen's right panel system provides context-sensitive controls for every aspect of your design.

Layer Panel (⌘⇧L)

Organize your artwork into logical layers.

  • Add/delete layers
  • Show/hide and lock layers
  • Reorder by dragging
  • Adjust layer opacity and blend modes

Paint Panel (⌘⇧P)

Control fill and stroke properties.

  • Fill: Color, gradient, or pattern
  • Stroke: Width, color, caps, joins, dashes
  • Shadow: Color, offset, blur
  • Opacity: Overall transparency

Grade Panel (⌘⇧G)

Create and edit gradients.

  • Linear and radial types
  • Add/remove color stops
  • Adjust angle and scale
  • Save gradient presets

Ink Panel (⌘⇧C)

Advanced color selection.

  • RGB, CMYK, HSB color modes
  • Color wheel and sliders
  • Pantone and system colors
  • Eyedropper sampling

Path Panel (⌘⇧O)

Boolean and path operations.

  • Unite: Merge shapes
  • Subtract: Remove top from bottom
  • Intersect: Keep overlap only
  • Exclude: Remove overlap (XOR)
  • Outline stroke, simplify, join paths

Font Panel (⌘⇧F)

Typography controls.

  • Font family and style
  • Size, leading, tracking, kerning
  • Alignment options
  • Convert to outlines

Panel Tips

  • Use keyboard shortcuts to switch panels
  • Double-click tab to minimize
  • Tab key hides all panels
