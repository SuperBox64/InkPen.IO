Interface Overview

The Logos InkPen interface is designed for efficient vector design work, with tools and panels organized for quick access.

Main Window Layout

The interface consists of: Menu Bar (top), Document Toolbar, Vertical Toolbar (left), Canvas Area (center), Right Panels, and Status Bar (bottom).

Interface Components

1. Menu Bar

Access all application commands through the menu bar:

  • Logos InkPen: Preferences, Default Tool settings
  • File: New, Open, Save, Import/Export options
  • Edit: Undo/Redo, Cut/Copy/Paste, Selection commands
  • Object: Arrange, Group, Transform, Path operations
  • Panel: Quick access to right panel tabs
  • View: Zoom, Display modes, Rulers, Grid, Snap options
  • Tools: All drawing tools with keyboard shortcuts
  • Window: Window management
  • Help: Documentation and tips

2. Document Toolbar

Quick access to frequently used commands:

  • Transformation Tools: Flip horizontal/vertical, rotate 90°
  • Corner Radius: Adjust corner radius for selected shapes
  • Close Path: Close open paths (⌘⇧J)
  • View Modes: Toggle between Color and Keyline view
  • Rulers: Show/hide rulers (⌘R)
  • Grid: Show/hide grid with snap (⌘')
  • Snap to Page: Fit artwork or selection to page
  • Export: Quick export to PNG

3. Vertical Toolbar (Left)

Primary drawing and selection tools:

  • Selection Tools: Selection, Direct Selection, Transform tools
  • Drawing Tools: Pen, Freehand, Brush, Marker
  • Shape Tools: Rectangle, Circle, Triangle, Polygon, Star
  • Specialty Tools: Text, Eyedropper, Gradient, Corner Radius
  • Navigation: Hand, Zoom tools
  • Color Swatches: Quick color selection grid at bottom
Tip: Long-press tool buttons to reveal tool variants and groups.

4. Canvas Area

The main drawing area where you create and edit your designs:

  • Rulers: Measurement guides along top and left edges
  • Grid: Optional alignment grid (toggle with ⌘')
  • Page Bounds: White rectangle showing document dimensions
  • Pasteboard: Gray area outside page for temporary objects
  • Selection Handles: Blue handles for transforming objects
  • Anchor Points: Path points visible with Direct Selection tool

5. Right Panel System

Context-sensitive panels organized in tabs:

Layer Panel (⌘⇧L): View and organize all layers, create, delete, rename, lock/unlock and show/hide layers.

Paint Panel (⌘⇧P): Fill settings, stroke settings, shadow effects, transform values, opacity and blending.

Grade Panel (⌘⇧G): Create and edit gradients, linear and radial types, add/remove color stops.

Ink Panel (⌘⇧C): Advanced color picker with RGB, CMYK, HSB modes, Pantone library, Apple system colors.

Path Panel (⌘⇧O): Boolean operations (Unite, Subtract, Intersect, Exclude), path commands.

Font Panel (⌘⇧F): Font family and style selection, size, leading, tracking, kerning, alignment.

6. Status Bar

Information display at bottom of window:

  • Zoom Level: Current zoom percentage
  • Cursor Position: X, Y coordinates
  • Selection Info: Number of selected objects
  • Document Units: Current measurement units

Customizing the Interface

Panel Management

  • Click panel tabs to switch between panels
  • Drag panel edges to resize
  • Double-click panel tab to collapse/expand
  • Use Panel menu to quickly jump to any panel

Keyboard Navigation

  • Space: Temporarily activate Hand tool
  • Tab: Hide/show all panels
  • ⌘0: Fit page in window
  • ⌘1: Actual size (100%)
  • ⌘+/−: Zoom in/out
